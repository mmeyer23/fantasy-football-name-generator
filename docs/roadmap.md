# Roadmap

## Milestone 1: Project Setup

- Select the web framework.
- Initialize the app.
- Add linting, formatting, and test setup.
- Create the initial UI shell.

## Milestone 2: Manual Player Entry

- Add active-player dataset.
- Build autocomplete search.
- Add selected-player chips.
- Add keyword input.
- Add clean/explicit mode control.

## Milestone 3: Deterministic Name Generation

- Build player-name normalization.
- Add curated phrase and pun templates.
- Generate clean results.
- Rank and de-duplicate suggestions.
- Add unit tests for generation behavior.

## Milestone 4: Explicit Mode and Filtering

- Tag templates and generated names by content mode.
- Add explicit-only templates.
- Add blocklist checks for clean mode.
- Test filter boundaries.

## Milestone 5: Screenshot Upload and OCR

- Add multi-image upload.
- Run OCR on uploaded screenshots.
- Extract candidate player names.
- Match candidates to active players.
- Add user confirmation/editing.

## Milestone 6: Data Refresh Workflow

- Choose roster data source.
- Add repeatable player-data update script.
- Validate generated player data.
- Document refresh steps.

## Milestone 7: Sharing and Polish

- Add copy, favorite, and regenerate controls.
- Improve result explanations.
- Add shareable links if appropriate.
- Improve responsive design and accessibility.
